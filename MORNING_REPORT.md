## 二、v3.0 新增功能详情

### 1. 面板扩宽至 640px + 座位网格对齐修复

- 面板宽度从 560px → 640px，内容呼吸感更强
- 彻底修复座位阶梯错位：每个舱位分段独立 CSS Grid，`--cols` 按实际布局设置
- 列字母按舱位独立计算 (公务 A-C-D-G, 超经 A-B-C-D-E-F-G, 经济 A-B-C-[过道]-D-E-F)

### 2. 多舱位分段系统

公务舱、超级经济舱、经济舱三段式布局，段间插入视觉分隔横幅：

| 舱位 | 宽体布局 | 窄体布局 | 噪声基线 | 分隔横幅色 |
|------|----------|----------|----------|------------|
| 头等/公务舱 | 1-2-1, Row 1-4 | 2-2, Row 1-3 | 58dB | 紫色渐变 |
| 超级经济舱 | 2-3-2, Row 5-8 | — | 64dB | 蓝色渐变 |
| 经济舱 | 3-3-3, Row 9+ | 3-3, Row 4+ | 70dB | 灰色渐变 |

- 公务舱座位默认绿色 "公务舱尊享座"，宽度 40px
- 超经舱座位 40px 宽度，独立 premium 样式
- 经济舱座位 38px 标准宽度
- 分隔横幅: `<div class="fp-cabin-divider">` 含渐变线 + 舱位名称徽章

### 3. 3D 粒子线框旋转地球 (纯 Canvas 正射投影，零 Three.js)

完全自主实现的 3D 渲染管线：

**渲染栈**:
- 星空粒子背景 (80 颗固定种子随机星点)
- 球体径向渐变光晕 (深空蓝底色)
- 纬度圈线框 (每 30°: 赤道/±30°/±60°)
- 经线线框 (每 30° 子午线)
- 6 大洲多边形轮廓线 (欧亚/非洲/北美/南美/澳洲/东亚)
- PEK (北京) / SYD (悉尼) 绿色高亮城市标记
- 大圆航线弧线 (虚线底色 + 亮色已飞段 + shadowBlur 辉光)
- 三角飞机图标沿弧线飞行 + 8 粒子尾迹
- Y 轴持续旋转 0.3°/帧 + 尾迹相位动画

**核心数学函数**:
| 函数 | 公式 |
|------|------|
| 经纬度→3D | `x=r·cos(φ)·sin(θ), y=r·sin(φ), z=r·cos(φ)·cos(θ)` |
| Y轴旋转 | `x'=x·cos(α)+z·sin(α), z'=-x·sin(α)+z·cos(α)` |
| 正射投影 | `sx=cx+x, sy=cy-y` (z≥0 正面裁剪) |
| 大圆插值 | slerp 球面线性插值，120 步采样 |

**底部信息栏**:
- `当前高空急流: W-240°/55kts`
- `ETOPS 180min 安全圈覆盖正常`
- 起止点 IATA 三字码标注

### 4. 噪声模型升级

噪声基线按舱位差异化，同舱位内按排位比例递增：

```
公务舱: base 58dB + 排位比例×(68-58) → 前排 58dB, 末排 68dB
超经舱: base 64dB + 排位比例×(74-64) → 前排 64dB, 末排 74dB
经济舱: base 70dB + 排位比例×(83-70) → 前排 70dB, 翼根 78dB, 尾排 83dB
```
